Cleaning Rules

20 rules applied in sequence during the pipeline

R1Midnight Rollover
Fix timestamps with hour 24:xx by rolling to next day at 00:xx. e.g., 28-02 24:15 becomes 01-03 00:15.
TSDatetime
R2Unicode Unit Fix
Normalize unicode µg/m³ to ASCII ug/m3. Prevents string mismatch in downstream joins.
UnitString
R3Status Trim & Case
Strip leading/trailing whitespace and normalize casing. " ok " → OK, " FAULT " → FAULT.
StatusString
R4Lat/Lon Validation
Parse the Lat_Lon string, validate coordinates fall within India (Lat 6–36°, Lon 68–98°). Fix semicolons, detect swapped lat/lon, flag impossible values like -999.
Lat_LonGeospatial
R5Negative Removal
Flag and discard physically impossible negative pollutant readings. Concentrations cannot be below zero.
PM2.5, SO2, NOxNumeric
R6Duplicate Merge
Detect duplicate rows by Plant_ID + TS composite key. Keep the first occurrence, log dropped duplicates.
Plant_ID + TSDedup
R7Zero/Span Calibration
Apply sensor drift correction: corrected = (raw - Zero_Drift) / Span_Mult. Uses values from sensor_master.
sensor_masterZero_Drift, Span_Mult
R8Spike Detection (Hampel)
Identify and cap extreme outliers (9999, 8500, etc.) using a Hampel filter on a rolling window. Replace spikes with the rolling median.
PM2.5, SO2, NOxOutlier
R9Gap Detection & Fill
Detect missing 15-min intervals by checking Record_ID gaps. Insert filler rows and forward-fill pollutant values from the last valid reading.
Record_ID, TSImputation
R10Maintenance Downtime
Cross-reference maintenance_logs time windows. If a raw reading falls during maintenance, null the pollutant values and force Status = MAINT.
maintenance_logsMaint_Start, Maint_End
R11Below Detection Limit
Replace BDL strings (<2.0, BDL, < 5.0) with LOD / 2 from sensor_master. Converts text to a usable numeric value.
sensor_masterLOD_PM25, LOD_SO2, LOD_NOx
R12Status Mapping
Map non-standard status strings to canonical values: offline/down → FAULT, maintenance → MAINT, Error 404 → FAULT.
StatusEnum
R13Plant-Stack Mapping
Validate every Plant_ID + Stack_ID combo in raw data exists in sensor_master. Flag orphaned records with no matching sensor.
sensor_masterPlant_ID, Stack_ID
R14Unit Conversion (mg/Nm3)
Convert mg/Nm3 to ug/m3 by multiplying the pollutant value by 1000. Then standardize the unit string.
Unit, PM2.5, SO2, NOxConversion
R15Source Type Tagging
Add a Source_Type column by joining on sensor_master. Tags each row as "Stack" or "Ambient" for threshold lookups.
sensor_masterSource_Type, Stack_ID
R16Double-Entry QC Check
Compare Lab_PM25_Entry1 vs Entry2 in manual_entries. If they differ by >1%, flag as QC_FAIL (likely decimal typo).
manual_entriesEntry1, Entry2
R17UTC → IST Conversion
Detect timestamps ending with UTC suffix and convert to Indian Standard Time by adding +5:30.
TSTimezone
R18PII Redaction
Regex scan Inspection_Notes for phone numbers (10-digit) and email addresses. Replace matches with [REDACTED].
manual_entriesInspection_Notes
R19Regulatory Threshold Check
Join regulatory_thresholds on Pollutant + Source_Type. Flag any cleaned reading exceeding the Legal_Limit_ugm3.
regulatory_thresholdsLegal_Limit_ugm3
R20Audit Trail Hash
Generate a SHA-256 hash for each cleaned row to create a tamper-proof audit column. Any post-cleaning modification will break the hash.
AuditAll columns
